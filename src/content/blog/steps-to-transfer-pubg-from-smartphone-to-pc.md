Transferring PUBG Mobile from your smartphone to your PC can save you time and internet data. By copying the game files directly, you can avoid re-downloading the large game on your computer. This guide will walk you through the step-by-step process of transferring PUBG Mobile to your PC using a USB cable, so you can enjoy the game on a larger screen without using up your mobile data.

## Steps

**1. Download PUBG Mobile on your Android device.**

Before beginning the transfer process, make sure you have PUBG Mobile installed and fully updated on your Android smartphone. Open the Google Play Store, search for **PUBG Mobile**, and install the game if you haven’t already. If you have the game installed, check for any available updates and install them.

**2. Connect your smartphone to your PC using a USB cable.**

Using your USB data transfer cable, connect your Android device to your computer. Typically, you’ll use the same cable that came with your phone or that you use for charging. Plug the smaller end into your phone, and the larger USB end into an available USB port on your PC.

**3. Open My Computer or This PC on your system and navigate to the Phone storage drive.**

On your computer, open up the File Explorer. Look for an icon representing your connected Android device. It may appear as an additional drive with your phone’s name. Double-click the icon to view your phone’s internal storage.

**4. Navigate to Phone>>Android>>Data>>com.tencent.ig and copy the folder to a new folder on your desktop.**

Within your phone’s storage, navigate to the Android folder, then open the Data folder. Find the folder named **com.tencent.ig** and copy it. Create a new folder on your PC’s desktop and paste the **com.tencent.ig** folder into it. This folder contains important PUBG Mobile game data.

**5. Navigate to Phone>>Android>>OBB>>com.tencent.ig and copy the folder to the same new folder on your desktop.**

Go back to your phone’s storage and open the Android folder again. This time, navigate to the OBB folder and look for another **com.tencent.ig** folder. Copy this folder and paste it into the same new folder you created on your desktop in the previous step. The OBB folder contains additional game resources.

**6. Start the Gameloop emulator on your PC and change the language to Chinese.**

Download and install the Gameloop emulator on your PC if you haven’t already. Launch Gameloop and navigate to the settings menu. Change the language to Chinese. This step is necessary for accessing certain features needed for the transfer process.

**7. Press F9 and open the browser app in the emulator.**

Within Gameloop, press the F9 key on your keyboard to bring up a hidden menu. Select the browser app from the menu to open it. If F9 doesn’t work, try Function + F9 or Shift + F9.

**8. Install ES File Explorer and give it root permissions.**

In the Gameloop browser, search for **ES File Explorer apk mirror** and download the app. Install ES File Explorer in the emulator and grant it root access when prompted. ES File Explorer will allow you to manage files within the emulator.

**9. Go to Data >> shared1 in ES File Explorer and find the OBB and Data folders copied earlier.**

Open ES File Explorer in the Gameloop emulator. Navigate to the **Data** folder, then open the **shared1** folder. This is where you’ll find the **com.tencent.ig** folders that you copied from your phone earlier.

**10. Copy both folders and paste them in Emulator storage>>Android.**

Select both the **com.tencent.ig** folders in the **shared1** directory and copy them. Navigate to the **Android** folder within the emulator’s storage and paste the copied folders here. If prompted, grant ES File Explorer permission to access this directory.

**11. Change the language back to English in Gameloop and start playing PUBG Mobile on your PC.**

Once the files are copied, close ES File Explorer and change the Gameloop language back to English in the settings menu. You should now see PUBG Mobile in your Gameloop game library. Click on the game icon to launch PUBG Mobile and start playing on your PC!

## Conclusion

By following these steps, you can easily transfer PUBG Mobile from your smartphone to your PC using a USB cable. This process allows you to enjoy the game on a larger screen without having to download it again using your internet connection. Keep in mind that you may need to update the game on your PC periodically to access new content and events. Enjoy fragging on the big screen!
