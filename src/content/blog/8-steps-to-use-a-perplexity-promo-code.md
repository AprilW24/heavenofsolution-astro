## Why Promo Codes for Perplexity Only Work in the Browser, Not the App

If you’re hoping to use a promo code on the actual Perplexity app, slow down! You can only use these codes on the website. The mobile app on Android or iOS won’t let you paste your code, or even find the promo code option. This means you’ll have to launch your phone’s browser (like Chrome or Safari), then head over to perplexity.ai. If you’re already using Apple or Google payment, wait until those subscriptions end before trying this. Promo codes are locked behind web-based payment to keep things fair and safe.

Most codes come from official emails, student programs, or partner offers. Expired or random codes won’t work at all, so make sure yours is genuine. If your subscription is through places like the Apple App Store or Google Play, you’ll need to let it expire before you apply any code in the browser. That’s a minor snag, but good to keep in mind before you get started.

## Step-by-Step Using Your Perplexity Promo Code on Mobile Web

1. Open your phone’s browser and type in *perplexity.ai*.

2. Log in to your existing account, or make a new one if you’re fresh.

3. Find the official referral or promo link you received via email, partner offer, or student deal.

4. Tap that promo link, if it’s legit, you’ll see a pop-up for the offer.

5. Choose your subscription plan.

6. If your promo code isn’t automatically filled, copy it from the pop-up.

7. Tap on “Add Promotion Code” at checkout and paste your code.

8. Hit “Subscribe” to seal the deal and start your discounted access.

Each step is super straightforward, no extra clicks needed. Promo codes only work once, and stacking isn’t allowed, so use your best deal code! 🎉

### What do I do if my promo code isn’t working?

Double check the code source, make sure it’s not expired, and refresh your browser. Still stuck? Reach out to Perplexity support.

### Can I use more than one promo code for my subscription?

No, you only get to use one code per billing cycle. Stacking or combining codes isn’t allowed.

### Why do I have to wait for my app-based subscription to end before using a promo code?

Because promo codes can only be used for web-based subscriptions. App Store or Google Play subscriptions need to expire first.

### What if I cancel my subscription after using a promo code?
