## How to Access and Manage Threads on Perplexity Android

Perplexity on Android keeps all your conversations neatly organized as threads, making it easy to pick up where you left off. You can tap the thread icon at any time to see your past discussions. Managing these threads is straightforward—just tap the one you want to revisit. This is especially handy if you want to compare different model responses or keep track of evolving questions. The interface is clean and user-friendly, so even if you’re new, you’ll find your way around in no time.

Once you’re inside a thread, you can scroll through the conversation history. Each message is clearly displayed, and you can see previous answers and edits. This makes it simple to review what’s been said and decide which part of the thread you want to rewrite. Thread management is a core part of the Perplexity experience, and on Android, it’s designed to be quick and intuitive, so you spend less time navigating and more time getting answers.

## Step-by-Step Guide to Rewriting a Thread with a Different Model

Ready to see what another model has to say? Here’s how you do it on Perplexity Android:

- Tap on the thread icon in the app to open your list of conversations.

- Click on the thread you want to rewrite and open it.

- Scroll down the thread or answer.

- Tap on the rewrite icon next, it’s usually a circular arrow or similar symbol.

- Select the model you want to use for the rewrite from the list.

- Tap the model you chose to confirm your selection.

- Wait for Perplexity to generate a new answer using the model you picked.

- Enjoy your new answer and see how it compares to previous ones.

## FAQs

### **Can I switch models before submitting a question on Perplexity Android?**

No, you need to submit your question first, then use the rewrite option to switch models.

### **How do I know which model was used for my answer?**

Currently, on Android, you can’t always see which model was used for each answer, but you can select the model before rewriting.

### **Can I keep both the original and rewritten answers?**

Yes, if you rewrite a message, it usually appears as a new entry in the thread, so you can compare both versions.

### **Are all models available to everyone?**

No, some models are only available to Pro subscribers, so you might not see every option unless you upgrade.

### **Can I edit my prompt before rewriting?**

Absolutely! You can add or change your text before hitting rewrite to get a more tailored answer.
